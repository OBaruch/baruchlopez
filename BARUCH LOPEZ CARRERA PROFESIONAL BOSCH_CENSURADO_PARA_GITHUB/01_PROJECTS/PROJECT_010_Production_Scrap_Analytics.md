# Analítica de Scrap de Producción

> Contenido anonimizado.

## Resumen
Solución de **analítica de scrap (merma) de producción** que **granulariza la información a nivel de centro de costo** (una versión previa solo llegaba a nivel agregado), sobre los datos de **una planta de manufactura**, con **ETL** y **Power BI**.

## Mi participación
Optimización del proceso para **mayor granularidad (por cost center)**, integrando datos de producción y construyendo el análisis automático.

## Tecnologías usadas
- **Alteryx** (ETL), **SQL Server**, **Power BI**.
- Fuentes: **SAP** y **un datamart interno que replica datos de SAP**.

## Actividades realizadas
- Pipeline de datos de scrap (actuals; escenarios), normalización y granularidad por cost center.
- Dashboard de scrap en Power BI.

## Impacto
- Mayor **granularidad y visibilidad** del scrap para decisiones de producción/calidad.
- **Ahorro relevante de horas-hombre** (probablemente cientos/mes) por automatización del análisis.

## Complejidad técnica
Media-alta: integración SAP/datamart → ETL → Power BI con granularidad fina.

## Habilidades / bullets para CV
- Desarrollé la **analítica de scrap de producción** con granularidad por centro de costo (ETL + Power BI), automatizando un análisis antes manual.
