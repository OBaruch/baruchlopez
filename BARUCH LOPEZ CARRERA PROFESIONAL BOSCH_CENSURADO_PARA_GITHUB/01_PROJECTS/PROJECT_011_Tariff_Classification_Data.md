# Habilitación de Datos para Clasificación Arancelaria

> Contenido anonimizado.

## Resumen
Iniciativa de **extracción y estructuración de datos de clasificación arancelaria** (fracción arancelaria) de comercio exterior: a partir de **fuentes oficiales del gobierno** (sistema público de clasificación + diario oficial), se construyeron **datos dimensionales** por niveles (capítulo, partida, subpartida, fracción) para alimentar después **un asistente de IA interno**.

## Mi participación
Investigación de fuentes, **scraping/extracción**, desarrollo de **múltiples scripts** para estructurar y correlacionar los datos, y preparación del dataset dimensional para su consumo posterior por un chatbot.

## Tecnologías usadas
- **Python** (scraping, procesamiento), **Excel**.
- Fuentes oficiales públicas (clasificación arancelaria + diario oficial).
- Salida estructurada para un **asistente/chatbot de IA interno**.

## Actividades realizadas
- Identificación de fuentes y extracción (scraping) de catálogos arancelarios.
- Estructuración en tablas + correlación con datos oficiales → **datos dimensionales** por niveles de la fracción.
- Preparación del dataset para alimentar un asistente de IA.

## Impacto
- Habilitación de datos para **clasificación arancelaria** (reduce trabajo manual; soporta cumplimiento aduanero).

## Complejidad técnica
Media-alta: scraping y normalización de catálogos normativos extensos y cambiantes; modelado dimensional.

## Habilidades / bullets para CV
- Desarrollé la **habilitación de datos para clasificación arancelaria** (scraping + estructuración dimensional con Python) a partir de fuentes oficiales, para alimentar un asistente de IA.
