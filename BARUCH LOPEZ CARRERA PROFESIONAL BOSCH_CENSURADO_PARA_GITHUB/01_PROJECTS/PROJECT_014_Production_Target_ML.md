> Contenido anonimizado.

## Resumen
Trabajo de **Machine Learning (regresión / forecast)** para **establecer targets de producción realistas por línea/producto**, a partir de datos de turno de manufactura (MES): piezas producidas, OEE, tiempo de ciclo, productividad.

## Mi participación

## Tecnologías usadas

- Fuente: datos de **un sistema de manufactura (MES)** en SQL Server.

## Actividades realizadas

- Matriz de tiempos de ciclo vs. producto.

## Impacto
