# Analítica de Costo de Fallas (Failure Cost)

> Contenido anonimizado.

## Resumen
Flujo de datos (**Alteryx**) que cuantifica el **impacto financiero del scrap / costo de fallas** en producción, combinando **un datamart interno (réplica de SAP)**, datos de calidad y tablas dimensionales, para reporte financiero y mejora operativa.

## Mi participación
Desarrollo del **pipeline Alteryx** (ingesta multi-fuente, limpieza, transformación node-by-node, salida) y su documentación técnica.

## Tecnologías usadas
- **Alteryx** (ETL), consultas SQL sobre **un datamart interno (Oracle, réplica de SAP)**, tablas dimensionales (Excel/CSV).
- **Power BI** (capa de visualización).

## Actividades realizadas
- Extracción de costos de falla y scrap (incl. órdenes de calidad), integración de reembolsos y mapeos dimensionales.
- Limpieza/transformación y construcción del dataset de failure cost.

## Impacto
- **Cuantificación del costo financiero del scrap** por unidad/causa → soporte a reporte financiero y mejora.

## Complejidad técnica
