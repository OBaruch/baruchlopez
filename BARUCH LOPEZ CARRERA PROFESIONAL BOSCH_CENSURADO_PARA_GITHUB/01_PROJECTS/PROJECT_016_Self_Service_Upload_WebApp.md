# Web App de Carga Self-Service (Flask)

> Contenido anonimizado.

## Resumen
**Aplicación web en Flask** que permite a usuarios de negocio **subir archivos Excel (por escenario, año y unidad de negocio)**, **validarlos** e **insertarlos en una base de datos** — un **uploader self-service** que alimenta el ecosistema de presupuesto/cockpits de planta.

## Mi participación
Desarrollo de la **app web (Flask)**: rutas de upload, validación de Excel con pandas, inserción a BD (SQLAlchemy), y material de capacitación.

## Tecnologías usadas
- **Python + Flask**, **Jinja2**, **pandas** (validación), **SQLAlchemy** (BD).

## Actividades realizadas
- Endpoint de carga con validación de formato; inserción a BD; capacitación a usuarios.

## Impacto
- **Autoservicio** de carga de datos con validación → reduce trabajo manual y errores de formato.

## Complejidad técnica
Media: app web con validación de datos e integración a BD; valor en habilitar a usuarios de negocio.
