# Participación: Automatización de Base de Datos (E-commerce)

> Contenido anonimizado.

## Resumen
Colaboración creando **scripts de automatización** para actualizar una base de datos de clientes en un contexto de **e-commerce**, usando orquestación con **Airflow (DAGs)**, conexiones **SFTP/FTPS** y extracción desde APIs.

## Habilidades relacionadas
Airflow (DAGs/Hooks), SFTP/FTPS, APIs, Python, SQL, data engineering.

## Relevancia
Demuestra **data engineering / orquestación de pipelines (Airflow)** e integración de sistemas.
