# Participación: Analítica Operativa (Cockpits / KPIs)

> Contenido anonimizado.

## Resumen
Conjunto de **iniciativas de analítica y data-pipeline de operaciones/controlling** entregadas como AI Engineer, que comparten stack (**ETL, SQL Server, Power BI, SAP/data-lake**) y dominio (operaciones de planta).

## Componentes
- **Cockpits operativos** — dashboards de seguimiento.
- **Seguimiento de activos en construcción (AUC)** — analítica financiera/controlling.
- **Bono de productividad** — analítica (Piezas/Horas) comparando datos de planta vs. reportes estándar de SAP.

## Línea de datos (alto nivel)
**SAP (reportes estándar) → datamart interno (réplica de SAP) → ETL → SQL Server → Power BI.**

## Habilidades relacionadas
Alteryx, SQL Server, Power BI, DAX, integración SAP/data-lake, Python.

## Relevancia
Demuestra **amplitud en analítica operativa y de controlling**, entregando múltiples soluciones de datos para distintas áreas.
