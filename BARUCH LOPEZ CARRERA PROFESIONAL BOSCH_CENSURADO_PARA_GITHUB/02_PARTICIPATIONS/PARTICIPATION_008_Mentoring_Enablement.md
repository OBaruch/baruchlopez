# Participación: Mentoría y Enablement

> Contenido anonimizado.

## Resumen
Actividades de **mentoría, enseñanza y habilitación técnica (enablement)**, que reflejan un perfil de **liderazgo técnico y transferencia de conocimiento**.

## Qué hice
- **Mentoría** dentro de un programa interno.
- **Material de capacitación** propio (ej.: tutorial interno de **GitHub Copilot** y de **Power BI**).
- **Educación en robótica/programación** a estudiantes (cursos-taller).

## Habilidades relacionadas
GitHub Copilot, Power BI, robótica/programación, mentoría técnica.

## Relevancia
**Technical leadership y enablement**: no solo construye soluciones, también **capacita y mentorea**.
