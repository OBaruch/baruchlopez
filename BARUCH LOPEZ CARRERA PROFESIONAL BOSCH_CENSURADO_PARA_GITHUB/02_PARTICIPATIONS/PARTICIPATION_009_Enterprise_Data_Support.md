# Participación: Soporte de Datos Empresariales (SAP) y Validación Documental

> Contenido anonimizado.

## Resumen
**Participaciones de soporte** que complementan los proyectos de datos: trabajo hands-on con **SAP** (transacciones de data dictionary, tablas y controlling), acceso/roles a sistemas, y una colaboración de **validación de documentos/facturas**.

## Componentes
- **SAP hands-on:** transacciones para diccionario de datos, tablas y clases de coste (CO); apoyo a la extracción de datos hacia la capa analítica.
- **Validación de facturas** (clasificación correcto/incorrecto) en una colaboración de equipo.

## Habilidades relacionadas
SAP (transacciones, tablas, CO), validación de documentos, Python.

## Relevancia
Demuestra **dominio del backbone SAP** que alimenta los proyectos de analítica.
