# Tecnología: Power BI

## Contexto
Capa de visualización/modelado del ecosistema de controlling de planta: modelos estrella, DAX, Power Query M, integración con SQL Server / OLAP.

## Dónde lo apliqué (por dominio)
Presupuesto y cockpits operativos de planta, costo de fallas, scrap, análisis de cobertura de portafolio (aftermarket).

## Nivel de dominio
**Uso avanzado** (modelado dimensional, DAX, Power Query M, integración multi-fuente, publicación en Power BI Service).

## Frase para CV
"Power BI (modelado estrella, DAX, Power Query M) integrado a SQL Server, SAP/data-lake y OLAP."
