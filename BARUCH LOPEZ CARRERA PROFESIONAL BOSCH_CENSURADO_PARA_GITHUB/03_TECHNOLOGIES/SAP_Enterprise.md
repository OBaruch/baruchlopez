# Tecnología: SAP / Sistemas Empresariales

## Contexto
Backbone de datos de los proyectos de planta: extracción desde **SAP** hacia un **data lake / datamart** y SQL Server. Trabajo hands-on con transacciones estándar y reportes.

## Dónde lo apliqué (por dominio)
Extracción de datos de costos/producción/materiales hacia la capa analítica; consulta de datos maestros; roles/acceso a sistemas.

## Nivel de dominio
**Uso operativo** (consulta de tablas/CO, lectura de reportes estándar, extracción); no desarrollo profundo de ABAP.

## Frase para CV
"Integración de datos SAP (reportes estándar de costos/producción/materiales) hacia data-lake y SQL Server."
