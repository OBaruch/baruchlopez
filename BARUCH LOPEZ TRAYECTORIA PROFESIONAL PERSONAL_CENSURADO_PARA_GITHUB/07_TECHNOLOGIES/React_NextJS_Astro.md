# React, Next.js, Vite y Astro

## Evidencia

- Next.js para Cyrus.
- Expo/React Native para Patrimonia.
- React/Vite para GAIA-1 y sitios comerciales.
- Astro para el portal personal.

## Nivel inferido

Uso operativo en varios tipos de producto.

## Señal de diseño

La selección de framework varía por caso: static-first en marca personal, Next.js en portal interno, Expo en móvil y Vite en landings/prototipos.
